# 3 Контрольная Фронтэнд и Бекэнд разработа

В рамках 3КР реализовано 5 практических заданий: 

- 13ПР: Реализован начальный каркас будущего простого оффлайн pwa приложения для ведения заметок, с использованием ServiceWorker. 
- 14ПР: Добавлен Web App Manifest (manifest.json), так называемый "паспорт" моего будущего pwa приложения, содержащий в себе мета-информацию для корректного отображения страницы в браузере, как полноценного нативного приложения.
        Также сделаны и загружени иконки для разных размеров экрана, обновлен ServiceWorker для кеширования иконок и файлов манифеста.
- 15ПР: Добавлен App Shell для мгновеной загрузки страницы на стороне пользователя, посредством кеширования файлов, которые учавствуют в каркасе приложения. Также был настроен локальный https, с помощью утилиты mkcert
        В качестве стратегии для получения статического контента (каркаса) страницы выбрана стратегия Cache First (программа сразу обращается к кешу), для динамического контента - Network First (Сначла пробуем перехватить запрос и только после неудачи, обращаемся к кешу).
        Также index.html разбит на about.html и home.html, где первично загрузиться index.html как каркас и потом подгрузит страницу из about.html
- 16ПР: Добавлены Web Socket через библиотеку Socket.io, для двустороенней открытой связи по TCP, для реализации push уведомлений о создании заметки, шде в тело самого уведомления передается заголовок и информация о том, что она была создана.
- 17ПР: Реализован функционал пуш напоминаний по заметкам, которые соединяются с средствами API для пуш уведомлений браузера, посредство Public VAPID ключа. Добавлен интерфейс для создания заметок с timestamp, который можно продлить нажав на кнопку продления в push уведомления.
        Также поверх материала практики было реализовано обновления timestamp в интерфейсе заметок, который отображает новое время если пользователь выбирает отложить push уведомление.
  
## Итоги третьей контрольной работы - современное PWA приложение для ведения заметок
Итогом 3 кр стало современное PWA приложение которое взаимодействует с браузером в открытом непрерывном TCP соединении, передает данные, хранит и получает их, взаимодействует с API браузеров для получения push уведомлений по заметкам, контролирует планирование, а также работает в офлайне или при закрытой странице.
